ETC 國道流量平台

本平台提供國道 ETC 交通流量資料查詢與視覺化功能。使用者可依門架、車種、時間區間等條件查詢通行資料，並透過地圖與統計儀表板快速掌握國道路網的流量變化。

開始查詢資料 查看統計儀表板

平台功能

資料查詢

依起點門架、終點門架、車種與時間條件查詢國道流量資料，並於地圖上呈現查詢結果。

地圖視覺化

使用互動式地圖呈現門架位置、路段方向與流量資訊，讓資料不只是一張表格。

統計摘要

提供日流量、週流量、月流量等統計資訊，方便快速觀察國道使用趨勢。

資料說明

資料來源

本平台使用交通部高速公路局發布之國道 ETC 開放資料， 主要包含車種通行量統計（M03A）與旅次路徑原始資料（M06A）， 並透過資料清理、彙整與視覺化方式提供查詢服務。

為降低官方資料後續校正所造成的差異，本平台僅提供截至查詢日前三日 已完成整理之資料；實際可查詢日期仍以系統資料庫收錄範圍為準。

資料來源： 交通資料庫 - 高速公路局
